改进 卡片左侧 3px 状态色带（绿=完成、紫=生成中、红=失败、灰=待生成），扫视即可辨别状态；hover 微浮起 + 阴影扩散；筛选栏独立成带状区域，视觉层级更清晰。

核心改进总结

  • 配置区全宽 + 内联导入：取消独立导入卡片，导入按钮放在配置标题行右侧，释放 25% 的横向空间
  • 新增进度总览条：环形图 + 分段进度条 + 分状态计数，用户一眼掌握全局进度
  • 结构化任务区标题：标题行（含批量操作）与筛选栏分离为两个水平带，信息层级更清晰
  • 卡片状态色带：左侧 3px 竖条按状态着色，扫视即可辨别状态分布
  • 卡片交互升级：hover 微浮动 + 阴影扩散，选中态背景 + 紫色边框 + 复选框高亮
  • 配置区 header/body 分离：header 固定展示，body 可折叠，折叠后只占一行，任务区获得更多视口
  • 筛选/分组共存一行：状态 Chip 左对齐、分组 Chip 右对齐，用竖线分隔，避免混淆
