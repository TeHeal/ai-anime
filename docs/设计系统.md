## 2. flutter_screenutil 尺寸适配

**设计稿基准**：1920×1080（`main.dart` 中 `ScreenUtilInit.designSize`）

**字号边界**：`FontScale`（`pub/const/font_scale.dart`）限制 `.sp` 缩放比例在 0.65～1.2 倍，避免小屏字号过小、大屏字号过大。

### 2.1 扩展方法

| 扩展 | 用途 | 示例 |
|------|------|------|
| `.w` | 宽度、水平间距 | `SizedBox(width: 24.w)` |
| `.h` | 高度、垂直间距 | `SizedBox(height: 16.h)` |
| `.r` | 圆角、通用尺寸 | `BorderRadius.circular(8.r)` |
| `.sp` | 字号 | `TextStyle(fontSize: 14.sp)` |

### 2.2 使用约定

1. **尺寸**：`SizedBox`、`EdgeInsets`、`width`/`height` 等使用 `.w`、`.h`、`.r`
2. **圆角**：`BorderRadius.circular(X.r)`，X 为数字或 `RadiusTokens.xxx`
3. **字号**：`AppTextStyles` 已内置 `.sp`，自定义时用 `fontSize: 14.sp`
4. **图标**：`Icon(..., size: 20.r)`
5. **面板宽度**：`Spacing.listPanelMinWidth.w`、`Spacing.reviewLeftWidth.w` 等

### 2.3 导入

```dart
import 'package:flutter_screenutil/flutter_screenutil.dart';
```

## 5. 禁止事项

- 禁止硬编码 `Color(0xFF...)`、`Colors.grey`、`Colors.white` 等
- 禁止硬编码 `BorderRadius.circular(8)`，应使用 `RadiusTokens.xxx.r`
- 禁止硬编码 `SizedBox(width: 24)`，应使用 `Spacing.xxx.w` 或 `24.w`
- 禁止硬编码 `TextStyle(fontSize: 14)`，应使用 `AppTextStyles.xxx` 或 `14.sp`

## 6. Lint 提示

### 6.1 项目已启用的相关规则

项目通过 `package:flutter_lints` 引入 Flutter 推荐规则集，其中与设计令牌相关的规则包括：

| 规则 | 说明 |
|------|------|
| `prefer_const_constructors` | 鼓励对可 const 的构造函数使用 `const`，减少 Widget 重建 |
| `prefer_const_constructors_in_immutables` | 对 `@immutable` 类优先使用 const 构造函数 |
| `prefer_const_declarations` | 声明时优先使用 `const` 而非 `final` |
| `prefer_const_literals_to_create_immutables` | 创建不可变对象时优先使用 const 字面量 |
| `use_full_hex_values_for_flutter_colors` | Color 使用 8 位十六进制（如 `0xFFFFFFFF`） |

### 6.2 需手动避免的硬编码模式（无现成 Lint）

Dart/Flutter 官方 linter 目前**没有**禁止硬编码颜色、尺寸的规则（如 `avoid_hardcoded_colors`），以下模式需开发者自行遵守：

| 禁止模式 | 正确用法 |
|----------|----------|
| `Color(0xFF...)`、`Colors.grey`、`Colors.white` | `AppColors.xxx`、`Theme.of(context).colorScheme.xxx` |
| `BorderRadius.circular(8)` | `BorderRadius.circular(RadiusTokens.xxx.r)` |
| `SizedBox(width: 24)`、`EdgeInsets.all(12)` | `Spacing.xxx.w`、`Spacing.xxx.r` 或 `24.w` |
| `TextStyle(fontSize: 14)` | `AppTextStyles.xxx` 或 `14.sp` |
| `Icon(..., size: 20)` | `Icon(..., size: 20.r)` |

代码审查时请重点检查上述模式。
