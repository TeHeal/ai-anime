# AI 漫剧工厂 — 人物小传格式说明

> 与剧本导入类似，人物小传采用 **Markdown 格式**，便于编剧编写和系统解析。
> 格式说明与 `90.剧本部分.md` 中的剧本格式配套使用。

---

## 一、用途

人物小传是编剧在交付剧本时，**可选**提供的角色设定文档。当提供人物小传时：

- 系统从人物小传中**解析**角色信息（外貌、性格、声音等）
- 减少 AI 创作性补全的工作量，提高角色设定准确性
- 角色名必须与剧本中的角色名**完全一致**，系统才能正确匹配

---

## 二、格式规范

### 2.1 顶层结构

```markdown
# 角色设定

## 角色名A
- **字段名**：字段值
- **字段名**：字段值
...

## 角色名B
- **字段名**：字段值
...
```

- 使用 `# 角色设定` 作为文档标题（可选，用于识别文档类型）
- 每个角色使用 `## 角色名` 作为二级标题
- **角色名必须与剧本中出现的角色名完全一致**（包括标点、空格）

### 2.2 字段定义

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| **外貌** | 是 | 角色的视觉描述，用于图像生成和分镜 prompt | 银白长发高束，身形修长，墨黑长袍，右眼角有旧疤 |
| **年龄** | 建议 | 具体年龄或年龄段 | 十七岁 / 少年 / 青年 |
| **性别** | 建议 | 男 / 女 | 男 |
| **身份** | 否 | 角色在故事中的身份/职业 | 青云剑派弟子 |
| **性格** | 建议 | 性格关键词或简短描述 | 冷峻沉稳，寡言重情 |
| **声音** | 否 | 声音特质描述，用于 TTS 音色选择参考 | 低沉清冷，语速偏慢 |
| **背景** | 否 | 角色背景/经历（可选） | 自幼习剑，师从青云真人 |
| **备注** | 否 | 其他补充说明 | 左撇子，惯用左手持剑 |

### 2.3 字段格式

- 字段名使用 `**加粗**` 包裹
- 字段名与值之间使用 `：`（全角冒号）或 `:`（半角冒号）分隔
- 值可跨多行，下一行缩进或使用列表延续

**示例**：

```markdown
## 顾白
- **年龄**：十七岁
- **性别**：男
- **身份**：青云剑派弟子
- **外貌**：银白长发高束，身形修长清瘦。身着墨黑色长袍，腰系银色剑穗。
  面容冷峻，剑眉星目，右眼角下有一道浅淡旧疤。左手常握青锋剑。
- **性格**：冷峻沉稳，寡言重情，表面冷淡实则内心炽热
- **声音**：低沉清冷，语速偏慢，言简意赅
```

### 2.4 与剧本的对应关系

| 人物小传 | 剧本 |
|----------|------|
| `## 顾白` | `**人物：顾白，林月**` 或 `顾白：（冷峻）今日便让你见识` |
| 角色名 | 必须完全一致，否则无法匹配 |

**未匹配处理**：
- 人物小传中有、剧本中无的角色 → 忽略（可能是配角或未出场角色）
- 剧本中有、人物小传中无的角色 → 按「纯剧本模式」由 AI 创作性补全

---

## 三、解析规则

### 3.1 角色分割

- 按 `## ` 开头的行分割，每块为一个角色
- 第一块若以 `# 角色设定` 开头，则跳过，从第一个 `## 角色名` 开始

### 3.2 字段提取

- 匹配 `- **字段名**：值` 或 `- **字段名**: 值`
- 值取冒号后至行尾的内容
- 若下一行以空格缩进开头，则续接到上一字段的值中

### 3.3 字段映射

| 人物小传字段 | 系统字段 |
|-------------|----------|
| 外貌 | `appearance` |
| 年龄 | `ageGroup`（需转换为年龄段：少年/青年/中年/老年） |
| 性别 | `gender` |
| 身份 | 可存入 `style` 或扩展字段 |
| 性格 | `personality` |
| 声音 | `voiceHint` |
| 背景 | 可存入扩展字段 |
| 备注 | 可并入 `appearance` 或单独存储 |

## 五、常见问题

### Q1：角色名有别名怎么办？

剧本中可能用「顾白」，人物小传中写「顾白（小白）」。建议人物小传中**仅使用剧本中的主名**，系统按主名匹配。

### Q2：外貌写得很简略可以吗？

可以。系统会结合剧本内容做补全。但外貌越详细，后续图像生成和分镜 prompt 质量越高。

### Q3：人物小传和剧本可以合在一个文件里吗？

当前版本要求**分开上传**。剧本按 `90.剧本部分.md` 格式解析，人物小传按本格式解析。合并格式可在后续版本支持。
