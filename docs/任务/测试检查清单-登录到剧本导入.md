## 一、API 测试（自动化，先执行）

cd anime_ai && go test -v -run "TestAPI" .

| 测试用例 | 说明 |
|----------|------|
| TestAPI_Health | 健康检查 200 |
| TestAPI_Login | 登录成功返回 token |
| TestAPI_Login_Fail | 错误密码返回 401 |
