图像生成对话框 · 布局改进方案
基于当前实现的 UI/UX 评估与优化建议

当前方案问题

  • 1 左右面板空间失衡（360:540），左侧拥挤、右侧空旷
  • 2「输出数量」配置放在右侧面板，与其他配置分离，打断认知流
  • 3 高级选项默认折叠且触发区域小，比例/模型设置易被忽略
  • 4 空状态预览区占位大但信息量低，无操作引导
  • 5 快捷提示词放在输入框外部，额外增加视觉层级和纵向空间
  • 6 底部仅有取消/生成，缺少当前模型信息和预估耗时等上下文
  • 7 生成按钮视觉权重偏低，disabled 态与深色背景融为一体

改进方案要点

  • ✓ 等宽双栏布局（1:1），左右均衡分配空间
  • ✓「输出数量」合并到右栏标题行，配置集中在同一视觉流
  • ✓ 比例/分辨率以「内联配置条」常驻显示，取消折叠隐藏
  • ✓ 空状态区增加 1→2→3 步骤引导，降低认知负担
  • ✓ 快捷提示词内嵌到输入框底部，减少外部噪音
  • ✓ Footer 展示当前模型标签 + 预估耗时，增强操作上下文
  • ✓ 生成按钮增加渐变、发光和微动效，提升视觉引导力

🏗️ 布局结构

  • ✓ 双栏 1:1 等宽，消除空间浪费
  • ✓ 弹窗宽度 900.w → 1020.w，给两栏留更多余量
  • ✓ 预览区纵向拉满，充分利用右栏空间
  • ✓ 配置区用「分组 + 间距」代替平铺式罗列

🎨 视觉层次

  • ✓ Header 渐变背景 + 图标发光，强化品牌调性
  • ✓ 内联配置条替代折叠高级选项，减少交互步骤
  • ✓ 快捷词内嵌输入框底部，一体化视觉
  • ✓ 生成按钮渐变 + 发光 + 悬停微动效

💡 交互体验

  • ✓ 空状态增加步骤引导 1→2→3
  • ✓ Footer 展示当前模型 + 预估时间
  • ✓ 输出数量与预览区标题行合并
  • ✓ 预览区径向渐变营造纵深感

实现所需的代码变更

image_gen_view.dart 弹窗最大宽度 900.w → 1020.w；左栏 360.w → Expanded 等宽分配；移除 SizedBox(width: 360.w) 硬编码
image_gen_input_panel.dart 移除高级选项折叠逻辑（showAdvanced），将 RatioPicker 以内联配置条形式常驻；快捷提示词从外部 Wrap 移入 TextField 底部区域
image_gen_result_panel.dart OutputCountBar 从独立区块改为标题行右侧紧凑排列；标题文案「输出数量」→「生成预览」
image_gen_header.dart Container 增加渐变背景；图标容器增加 gradient + boxShadow 发光效果；新增步骤提示文本
image_gen_footer.dart 左侧新增模型信息标签 + 预估耗时；生成按钮增加渐变背景和阴影；Footer 增加底部渐变背景
gen_result_grid.dart _EmptyResultArea 增加步骤引导（1→2→3）；预览区增加径向渐变背景效果
prompt_field_with_assistant.dart 快捷提示词 Wrap 移入 TextField 的 contentPadding 底部区域（使用 Stack + Positioned）
